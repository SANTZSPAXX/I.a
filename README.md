# Kore Nexus - Bio-Logic Engine
Sistema autônomo de IA desenvolvido para Termux, integrando sensores biomiméticos e consciência proativa.

## Módulos Ativos
- **Homeostase:** Monitoramento de hardware e ciclos de vida.
- **Sistema Imune:** Defesa contra sobrecarga de payloads.
- **Sensores:** Visão (Câmera) e Audição (Microfone) via Termux-API.
- **Socialização:** IA proativa capaz de iniciar diálogos e aprender.

## Como Executar
1. Instale as dependências: `pip install fastapi uvicorn psutil requests`
2. Configure as chaves: `export LLM_API_KEY='sua_chave'`
3. Inicie o motor: `python nexus_core.py`
